Authen Gate is a cutting-edge security solution designed to protect sensitive data and devices by leveraging geo-fence technology. This project aims to create an added level of device security by restricting login access to hardware authentication devices based on GPS boundaries. By using GPS coordinate restrictions preset by the user, Authen Gate physically disables USB data transmission, ensuring that unauthorized access is prevented, and data remains secure.

Target Audiences:

Companies with Telecommuting Employees:

Authen Gate provides an ideal security solution for companies with remote workforces, allowing them to limit employee workspaces to pre-designated safe and approved locations. This ensures that sensitive company data is only accessed within secure environments.

Traveling Workers:

For professionals who frequently travel, Authen Gate offers an additional layer of security, ensuring that work devices cannot be accessed during transit. This minimizes the risk of data breaches in the event of theft or loss.

Incident Response Teams:

Authen Gate can help incident response teams safeguard sensitive information and proprietary equipment by preventing unauthorized access. By restricting device access to specific geographical locations, the solution helps protect valuable assets and maintain operational integrity.

General Users:

For everyday users, Authen Gate provides an added level of security in the event of theft or loss. By restricting access to their devices based on GPS boundaries, users can have peace of mind knowing their personal data remains secure.

Conclusion

In conclusion, Authen Gate is a revolutionary security solution that addresses the needs of a diverse range of users, from remote workers to incident response teams. By leveraging geo-fence technology and GPS coordinate restrictions, Authen Gate provides unparalleled protection against unauthorized access, ensuring that sensitive data and devices remain secure.

Technical Fields:

The Authen Gate project encompasses multiple technical fields, including Network Security, Robotics/Embedded Systems, Software Engineering, and Digital Maker/Fabrication. Network Security handles hardware authentication, geo-fence technology, and data encryption. Robotics and Embedded Systems are responsible for the prototype device's wiring, final PCB design, and construction. Software Engineering creates the scripts loaded onto the Arduino Nano Every devices and develops the user interface and open-source FIDO2 authentication. Finally, Digital Maker and Fabrication skills, such as Arduino development, 3D modeling, and 3D printing, are crucial for the project's final production.

Background Information:

The inspiration for Authen Gate stems from the rise of remote working and criminal activities since the Covid-19 pandemic. Many people and companies have struggled with the security requirements for remote workers. This project aims to fill the security gap by adding an extra layer of protection, ensuring that even if the device and authentication device are stolen together, access will be disabled once the devices are outside the predetermined boundaries.

Prior Art Disclaimer:

The project currently uses authentication resources from a Yubico YubiKey 5 Series device for login functionality. In the next phase, the YubiKey will be replaced with open-source technology.
