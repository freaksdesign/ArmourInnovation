Nessus

Nessus is a vulnerability scanner and port scanner software that is available in both free and paid versions for multiple operating systems. It offers 26 scan variations, easy configurability, built-in scheduling, email notifications, and custom policies and rules. Nessus can be installed on Linux by signing up for a token, downloading the .DEB, manually installing it, starting the service, and starting the final install through the local host. Once installed, Nessus is ready to scan and offers a live overview of the results.

AIDE

AIDE (Advanced Intrusion Detection Environment) is a free and open-source file and directory integrity checker that uses hashing algorithms to create a database of file and folder hashes. AIDE supports multiple hashing algorithms and can perform both manual and automatic file integrity checks. It supports multiple file system types and can monitor changes to file size, mtime, atime, ctime, permissions, users, and groups and send email change reports. The configuration of AIDE can be customized with various hash types, custom rules, cron job intervals, log rotation, and file compression types. AIDE is available on multiple Linux OS via package installer.

Lynis

Lynis is a fast security auditing and hardening tool that is available for most Unix flavors. It is a free and open-source tool that offers automated security auditing, daily system health scans, compliance testing for ISO27001, PCI-DSS, and HIPAA, vulnerability detection, and configuration issue detection. Lynis can be installed for free via package manager, Git command install, website download, or Homebrew for MacOS. It also offers an enterprise version that can be used as a cloud software-as-a-service or self-hosted.

The author received feedback on how to improve the attack and acknowledged that it was a mock attack for educational purposes only.
